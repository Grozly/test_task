Для работы с приложением необходимо.
1) Запустить локальный сервер командой - python manage.py runserver
2) Перейти по ссылке http://127.0.0.1/admin/ 
и пройти авторизацию, по умолчанию
логин - admin
пароль - admin
3) В панели администратора добавлен раздел deals, переходим в него.
4) Открываем новую вкладку и переходим по адресу http://127.0.0.1/api/upload_file/
5) В HTML-Form внизу страницы добавляем .csv файл с сделками deals.csv или dealsfull.csv
6) Файл будет обработан и все данные добавяться к нам в базу данных, 
также сразу получим вывод информации в GET.
7) Обновляем раздел deals в панели администратора, 
видим все данные перенесены из .csv файла в базу данных.
